# customer_behaviour_analysis
Data analytics project showcasing customer behaviour analysis using Python,MYSQL and Tableau.

## Overview

This project focuses on analyzing customer shopping behavior to identify purchasing patterns, customer preferences, and sales trends.

The project follows an end-to-end data analytics workflow, including:
- Loading and analyzing data using Python
- Performing Exploratory Data Analysis (EDA)
- Cleaning and preparing data
- Running SQL queries using MySQL
- Building an interactive Tableau dashboard

The goal of this project is to uncover meaningful insights about customer purchases, product performance, and shopping trends that can support business decision-making.

## Dataset

**Dataset Name:** Customer Shopping Behaviour Dataset

The dataset contains customer transaction and demographic information, including:

- Customer demographics
- Age and gender information
- Product categories
- Items purchased
- Purchase amount
- Purchase frequency
- Review ratings
- Shipping type
- Payment methods
- Discount usage
- Subscription status

Dataset Size: 1000

## Project Workflow

### 1. Data Loading & Exploration (Python)

- Imported the dataset using Pandas
- Examined dataset structure and data types
- Checked for missing values and duplicate records
- Reviewed statistical summaries
- Identified important customer and purchase attributes

---
### 2. Data Cleaning & Preprocessing

Data preparation steps included:

- Handling missing values
- Removing duplicate records
- Standardizing column names
- Correcting data formats
- Creating calculated fields where required
- Preparing a clean dataset for analysis

The cleaned dataset was then loaded into MySQL for further analysis.

---

### 3. Exploratory Data Analysis (EDA)

Performed analysis to understand customer purchasing behavior.

Key areas explored:

- Customer purchase distribution
- Top-selling products and categories
- Revenue patterns
- Customer segmentation
- Shopping frequency trends
- Relationship between discounts and purchases
- Customer ratings and product performance

Visualizations were created using Matplotlib and Seaborn to identify trends and patterns.

---
### 4. SQL Analysis Using MySQL

The cleaned dataset was imported into MySQL and analyzed using SQL queries.

Business questions explored:

- Which products are purchased the most?
- Which categories generate the highest revenue?
- What are the most common payment methods?
- How does customer behavior vary by age group?
- Which customers contribute the most revenue?
- Do discounts influence purchase behavior?
- Which seasons have higher sales?

SQL concepts used:

- SELECT statements
- Filtering using WHERE
- Aggregations
- GROUP BY
- ORDER BY
- CASE statements
- Subqueries
- Common Table Expressions (CTEs)

---
## Tableau Dashboard

An interactive Tableau dashboard was created to visualize key business insights.

Dashboard highlights:

- Total revenue and customer KPIs
- Sales by product category
- Top purchased items
- Customer demographics
- Seasonal purchase trends
- Payment method analysis
- Customer rating insights

Users can interact with filters to explore different customer segments and purchasing patterns.
---

## Key Results & Insights

The analysis provided insights into:

- Customer purchasing patterns and preferences
- Best-performing products and categories
- Revenue-driving customer segments
- Seasonal shopping trends
- Payment and shipping preferences
- Factors influencing purchase decisions

These insights can help businesses improve marketing strategies, inventory planning, and customer engagement.
